# 🥬 Applifey - Web-App Store for Linux

**Applifey** is a simple, lightweight App Store that turns your favorite web applications (like Microsoft 365, iCloud, WhatsApp Web, or Netflix) into real, native Linux desktop apps with just one click. 

No more cluttered browser tabs—launch your favorite web tools directly from your Linux system menu!

---

## ✨ Features

* 🌐 **Cloud Sync:** New apps are automatically added to the store worldwide directly from the GitHub cloud.
* ➕ **Custom Apps:** Link and install any website you want as a private desktop application.
* 📦 **Built-in App Manager:** Easily manage and completely uninstall your web apps with a single click in the settings.
* 🦊 **Browser Choice:** Fully supports Brave, Chrome, Chromium, and Firefox.

---

## 🚀 Download & Installation

*Note: Applifey is currently in active development.*

Once fully released, Applifey will be available as a standalone **AppImage**. You will not need to install Python or any extra software. Just download the file, make it executable, and double-click to run!
